# Multiplication Quest: The Times Table Trials

A complete GitHub Pages-ready multiplication fluency game for math facts 1–10.

## Features

- Quest map with unlockable lands
- Boss battles
- Basic story and lore
- Points, coins, streaks, lives, badges, and rewards
- Powers: Double Points, Freeze, Shield, Hint
- Levels of difficulty from facts 1–10
- Local leaderboard saved in the browser
- Teacher / Scores screen
- CSV export for scores
- Mobile-friendly layout
- No backend required

## Files

- `index.html`
- `style.css`
- `script.js`
- `README.md`

## How to Host on GitHub Pages

1. Create a new GitHub repository.
2. Upload all four files to the root of the repository.
3. Go to Settings → Pages.
4. Under Source, choose Deploy from a branch.
5. Choose branch: `main` and folder: `/root`.
6. Click Save.
7. Wait for GitHub to publish the site.
8. Open the live GitHub Pages link.

## Important Note About Leaderboards

This version uses browser storage. Scores save on the device where students play. This is safest and easiest for GitHub Pages because GitHub Pages does not provide a database.

For a true class-wide online leaderboard across different devices, connect the game to Google Sheets, Firebase, or another database service.
