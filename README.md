# Multiplication Quest: Math Facts 1–10

An interactive browser game for multiplication fluency. Students answer facts, earn points, collect coins, unlock badges, and move through levels of difficulty.

## Features

- Facts from 1–10
- Easy, Medium, Hard, and Boss levels
- Practice mode with no timer
- Points, coins, streak bonuses, and badges
- Local leaderboard saved on the device/browser
- Works on GitHub Pages with no backend

## How to Use on GitHub Pages

1. Create a new GitHub repository.
2. Upload these files to the repository root:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose the `main` branch and `/root` folder.
6. Click **Save**.
7. Open the GitHub Pages link after it finishes publishing.

## Classroom Note

The leaderboard uses browser local storage. It saves scores on the same device only. For a shared classroom leaderboard, have students play on one classroom device or add a Google Form/Sheet backend later.
